Principais comandos do terminal:
	(Para git)
git init - inicia um repositório git
git status - mostra se há arquivos não comitados
git add . - prepara arquivos para serem comitados
git commit -m "comentários"
git branch -M main (caso queira alterar o novo padrão da master)
git branch -b novo (caso queira adicionar uma o nova branch)
git checkout <nome-do-branch> ( alterna entre branchs)
git merge <nome-do-branch> ( Funde as branchs em uma só)
git remote add origin htps: ... (adiciona o caminho de destino)
git push origin <nome-do-branch> (empurra o conteúdo para o caminho escolhido)
git pull origin <nome-do-branch> (Puxa os arquivos do repositório)
git clone htps: ... ( Clona o repositório de uma fonte especifica indicada)

	(Para terminal)
clear ( Limpa o Terminal)
ls (lista os conteúdos dentro de uma página)

	(para Java)
public static void main(String[] args) --> Atalho <psvm>
System.out.println("") --> Atalho <sout>
